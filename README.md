# Airbnb Listings Dashboard 🏠

## Objective
Analyze Airbnb listings across 8 cities worldwide 
to understand pricing, host performance and location trends.

## Dataset
- 100 listings across 8 cities
- Cities: Mumbai, Delhi, Bangalore, Chennai, 
  London, Paris, Barcelona, New York, Tokyo, Singapore
- 15 columns | 6 countries

## Tools Used
- Power BI Desktop
- Power Query — Data Cleaning
- DAX — Custom Measures

## DAX Measures
- Total Listings
- Avg Price Per Night
- Avg Review Score
- Total Reviews
- Superhost Count

## Key Insights
- Tokyo & Singapore most expensive cities
- 50% hosts are Superhosts
- Entire home/apt most popular room type
- Higher price = Higher review score trend

## Dashboard Pages
- Page 1: Overview & KPIs
- Page 2: Location & Price Analysis
- Page 3: Host & Rating Performance
